Poker Stats Data (JSON)

Rådata för pokerstatistik som används på poker-sidan.

JSON-struktur: Array av Event-objekt

Array ([])
— Representerar alla spelade event. Varje element i arrayen är ett Event-objekt.
Event-objekt ({})
— Representerar en enskild pokerkväll.
  • id: Number - Unikt ID för eventet.
  • date: String - Datum i formatet YYYY-MM-DD.
  • location: String - Plats där eventet spelades.
  • buyIn: Number - Standard insats per spelare.
  • players: Array - En lista över alla deltagare i eventet.
Player-objekt ({})
— Representerar en enskild spelares resultat för kvällen.
  • name: String - Spelarens namn.
  • placement: Number - Spelarens placering (1 är bäst).
  • winnings: Number - Valfritt. Uttaget belopp i kontanter/pris om standard inte följs: Vinnaren: totalPot - buyIn, tvåan: buyIn
  • extraBuyIns: Number - Valfritt. Antal re-buys/extra insatser utöver den initiala `buyIn`.
