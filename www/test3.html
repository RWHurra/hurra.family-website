Förklaring av Grid-lösningen:

Denna lösning använder CSS Grid för att placera Element 1 i en dedikerad mittkolumn. Detta garanterar att **Element 1:s mittpunkt** alltid ligger i containerns mitt.

  1. Containern får display: grid;.
  2. Rutnätet definieras som 1fr auto 1fr;, där auto är mittkolumnen som reserveras för Element 1.
  3. Element 1 placeras i kolumn 2 (mitten).
  4. Element 2 placeras i kolumn 3 (höger) och dras till Element 1:s kant med justify-self: start;.

För **UL/LI-exemplet** (Test 3) appliceras .grid-container direkt på <ul>, och extra CSS-regler (.ul-list-reset) tar bort listpunkter och standardmarginaler/padding.
