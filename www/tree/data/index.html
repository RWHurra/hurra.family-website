Family Tree Data (JSON)

Rådata för släktträdet som används för att bygga visualiseringen på trädsidan.

JSON-struktur: Array av Person-objekt

Array ([])
— Innehåller alla personer i databasen. Varje person är ett unikt objekt med ett unikt ID.
Person-objekt ({})
— Innehåller biografisk data och relations-ID:n.
  • id: String - Unikt ID (används som referens).
  • firstName: String - Förnamn.
  • lastName: String - Efternamn.
  • birthDate: String - Format YYYY-MM-DD (eller YYYY-??).
  • deathDate: String | null - Dödsdatum om tillämpligt.
  • birthCountry: String - ISO-kod för födelseland (t.ex. "SWE").
  • residenceCountry: String - Nuvarande hemland.
  • citizenships: Array[String] - Lista på medborgarskap.
Relationer (Referenser)
— Dessa fält mappar till id på andra objekt i samma array.
  • motherId: String | null - ID till modern.
  • fatherId: String | null - ID till fadern.
  • spouseId: String | null - ID till partner/make/maka.
  • childrenIds: Array[String] - Lista med ID:n till barn.
